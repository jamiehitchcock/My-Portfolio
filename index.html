Some of my work.

Here are some of the projects that I have created to test my skills. You can access these from the image and also view the code on my github.

Real-Time Stock Market Checker

JavaScript, APIs.

I created a stock market checker that will allow you to enter a symbol for a stock and view realtime information on its perfomance. This takes the value of the symbol entered and fetches the relevant stock from api (StockData). Try AAPL, GOOG, PG or any other stock.

Internation Space Station Tracker

JavaScript, Leaflet.js, APIs

This takes fetches data from an api (WhereTheIss), such as the coordinates and velocity. This data is then used in leaflet.js to display an icon moving on the map.

Roman Numeral Converter

JavaScript, Algorithms

If you enter a number, the algorithm will convert this into its roman numeral equivalent and display it back.

Fun Animated Character

CSS, Animation

Digital Clock

JavaScript

This uses the JavaScript Date Object to display the current date and time. I then added the functionality to convert this to the day names too. I used similar code to display the day name at the end of the contact me section on this website.
